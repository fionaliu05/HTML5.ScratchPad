About

This is an example of a SPA app with Modals using Angular JS Framework.

The following JavaScript Frameworks were used:

  • Typescript

  • JQuery 2.1.4

  • Bootstrap 3.3.6

  • Angular JS - 1.4.7

  • Angular-Animate JS - 1.4.8

  • Angular-Route - 1.4.7

  • Angular-Material - 1.4.7

  • Angular-UI-Bootstrap - ~0.12.0

  • NodeJS - 4.2.3

The folling packagers were also used:

  • Bower - 1.7.1

  • Gulp - 3.9.0

  • npm - 2.14.7

  • tsd - 2.14.7
